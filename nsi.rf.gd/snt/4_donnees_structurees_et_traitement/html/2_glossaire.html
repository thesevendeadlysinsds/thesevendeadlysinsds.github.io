2 - Glossaire

Une donnée

Définition:

Valeur ou ensemble de valeurs pouvant prendre plusieurs formes (nombre, chaînes de caractères) et permettant de décrire un ou plusieurs objets.

En d'autres termes:

C'est une information sur un objet ou une personne. Pour une classe d'élèves de 2de, une donnée pourrait ainsi correspondre au prénom d'un élève.

Une base de données

Définition:

Entité stockant des données structurées et permettant leur traitement à l'aide d'algorithmes.

En d'autres termes:

Une base de données est une sorte de réservoir de données que l'on peut récupérer et étudier. Le secrétariat d'un lycée collecte et utilise ainsi de nombreuses informations qui concernent les élèves, les professeurs, les classes, etc. Toutes ces données constituent la base de données de l'établissement.

Les données structurées

Définition:

Ensemble de données stockées sous la forme d'une table regroupant de multiples objets selon un ou plusieurs descripteurs.

En d'autres termes:

Les données peuvent être structurées en tableau regroupant des informations sur plusieurs objets selon leur caractéristiques. Pour la classe de 2nd, il peut ainsi s'agir d'un tableau regroupant plusieurs informations sur les élèves de la classe, comme leur prénom, leur nom, leur date de naissance, etc...

Le cloud

Définition:

Service de stockage de données à distance via internet.

En d'autres termes:

Le cloud est un service qui permet de sauvegarder et d'accéder à ses dossiers sur Internet. Le cloud, ou nuage, peut être associé à un vaste espace dans lequel les données informatiques sont envoyées sans que l'on sache vraiment où elles se trouvent. C'est ce qui permet aux enseignants de faire l'appel en ligne, par exemple.

Les métadonnées d'un fichier

Définition:

Ensemble de données décrivant les caractéristiques d'un fichier, sans être spécifiquement liées à son contenu.

En d'autres termes:

Ce sont les informations qui décrivent un fichier. Par exemple, une fois que l'on a saisi toutes les informations des élèves de la classe dans un tableur, le nom, la date de création ou le type de fichier constitueraient des métadonnées.
